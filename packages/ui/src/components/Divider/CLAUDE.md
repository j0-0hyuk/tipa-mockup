# Divider

구분선 컴포넌트입니다.

## 사용법

```typescript
import { Divider } from '@docs-front/ui';
```

## 예시

```tsx
// 가로 구분선
<Divider />

// 세로 구분선
<Divider direction="vertical" />
```

## Props

| Prop | Type | 설명 |
|------|------|------|
| `direction` | `'horizontal' \| 'vertical'` | 방향 (기본: horizontal) |
